# Fresh Tomato

Pomodoro app written in Deno Fresh

## Usage

```
deno task start
```

## Architecture

- [Preact Signals](https://preactjs.com/guide/v10/signals/) for global state management
- Complicated CSS was placed into `static/styles.css`, particularly around theming
- [fresh_postcss](https://deno.land/x/fresh_postcss) plugin
